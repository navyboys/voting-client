# voting-client

1. Install packages

  ``` bash
  npm install
  ```

1. Run test

  ``` bash
  npm run test
  ```

1. Start `webpack-dev-server`

  ``` bash
  webpack-dev-server
  ```

1. Start server in another terminal

  - [Voting Server](https://github.com/navyboys/voting-server)

1. In browser, access
  - `localhost:8080/` for voting
  - `localhost:8080/#/results` for results
